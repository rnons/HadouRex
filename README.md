# HadouRex

## How-To

<pre>
runhaskell dourex.hs
</pre>

You may want to compile it.

<pre>
ghc --make dourex.hs
</pre>

## Commands

<pre>
./dourex
./dourex listen cid
./dourex search keywords
./dourex hot
./dourex trending
</pre>

When playing, press `h` to see a full list of available controls (of mpg123).
